# Como iniciar um projeto

FLUXO DE TRABALHO AUTOMAÇÔES

- planejar
- criar ( em fases)
- Testa (em fases)
- sobe/ faz deploy
- cobra por acesso (usando licenças ou algo similar)
- Mensalidade

Criar um arquivo txt para a demanda

Tenha algo + ou - pronto antes de iniciar um projeto no modo plan do claude code

pesquisar tudo no claude code é muito caro, tenha uma conta secudaria gratuita ou use o gpt para fazer pesquisas de tecnologia ou ideias

descrever as regras de negocio (nao precisa ser todas), alguma tecnologia que vc pesquisou, descrever a arquitetura de pastas, descrever valores e peça sugestoes.

back end, front end e onde armazenar

Solicitar um arquivo [plan.md](http://plan.md) depois de planejar o projeto.

para implementar o [plan.md](http://plan.md) peça para o claude quebrar o projeto em etapas e a cada etapa concluida ele teste usando o Playwright MCP e peça meu feedback.

alem disso, cria um plano de como esse sistema deve ser implementado, ou seja, um passo a passo de como subir esse sistema para uma VPS de $5 da Digital Ocean, por exemplo.

depois pedir para ele criar um sistema de licenças
