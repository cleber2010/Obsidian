# Skills

Intruções para que o claude seja mais eficiente e determinada tarafe

ex

criar uma landing page elegante e bonita, ele vai utilizar os padroes de ui e ux do mercado a partir de intruções especificas em um arquivo .md

Estrutura e como usar:

- Criar uma pasta .claude
- crair uma pasta skills
- colocar a skill dentro dessa pasta
- referenciar o nome da skill no prompt
