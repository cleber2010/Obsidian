# Como Vender Automações

## Onde ter presença

- Upwork
- Workana
- Freelancer
- LinkedIn
- Instagram / TikTok
- Kawai

---

## Tipos de entrega

### Sites
- Hospedar no Netlify
- Criar domínios
- Definir precificação

### Sistemas / Automações
- Criar executáveis/instaladores
- Sistema de licenças
- Cobrar por acesso ou mensalidade

### Sistemas Web
- Hospedar sistemas web (Netlify, Vercel)
- Bancos de dados hospedados (Supabase)
- VPS para mais controle

---

## Modelos de negócio

- **Automação de processos** na empresa onde o cliente trabalha (modelo consultoria)
- **Produto SaaS** — sistema recorrente com mensalidade
- **Licença por uso** — cliente paga por acesso, sem mensalidade
